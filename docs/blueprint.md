# **App Name**: Carton Sheet Calc

## Core Features:

- Home Page: A welcoming landing page with a brief description of the application.
- Article Data Display: Display static data from 'data_artikel.json' in a sortable and searchable table.
- Price Calculator: Calculate the price of a sheet with dynamically addable rows.
- MOQ Calculator: Calculate the Minimum Order Quantity with dynamically addable rows.
- Tonnage Calculator: Calculate total tonnage with dynamically addable rows and total sum.
- Tonnage Logic (Single Wall): JavaScript functions to calculate tonnage for single wall sheets.
- Price and MOQ Calculation Logic: JavaScript functions to calculate price and MOQ based on provided data.

## Style Guidelines:

- Primary color:  (#ffdc5f) to reflect professionalism and precision.
- Background color:  (#Fffdf0), close in hue to indigo, but much lighter and more desaturated.
- Accent color: A slightly more (#001840), (#102a71), and glasses effect like windows 11 or apple ios latest version,  for interactive elements. text auto color to white if viewed at  color dark, and text auto color to black if viewed at light color
- Body and headline font: 'Inter' (sans-serif) for a clean, modern and neutral feel.
- Use simple, professional icons for calculations and data display.
- Excel-like dynamic tables with smooth transitions and interactive columns.
- Subtle cursor and page transition animations for a modern feel.
